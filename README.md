# Portfolio Website

A modern portfolio website built with React, TypeScript, and Three.js.

## Technologies Used

- React
- TypeScript
- Three.js
- Framer Motion
- Tailwind CSS
- Vite

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```

## Build

To build the project for production:
```bash
npm run build
```
